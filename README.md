# GenAI e Dados (DIO Bootcamp) / GenAI & Data (DIO Bootcamp)

## Português (Brasil)

**Projeto:** GenAI e Dados — bootcamp DIO criado por Bradesco

**Descrição:** Pequeno projeto didático para aprender conceitos iniciais de inteligência generativa e tratamento de dados. Contém modelos básicos (`user`, `destino`, `reserva`), utilitários e conexão SQLite.

**Estrutura:**
- `models.py` — classes de domínio
- `connect.py` — helper de conexão SQLite
- `config.py` — configuração do caminho do banco (`DATABASE_DIR`)
- `utils.py` — utilitários

**Como usar:**
1. Configure um ambiente Python e instale dependências em `requirements.txt` (se necessário).
2. Execute scripts ou importe módulos conforme seu fluxo de trabalho.


## English

**Project:** GenAI & Data — DIO Bootcamp created by Bradesco

**Description:** Small educational project to learn introductory generative AI and data handling. Includes simple domain models (`user`, `destino`, `reserva`), utilities, and an SQLite connection helper.

**Structure:**
- `models.py` — domain classes
- `connect.py` — SQLite connection helper
- `config.py` — configuration for `DATABASE_DIR`
- `utils.py` — utility functions

**How to use:**
1. Create a Python environment and install dependencies listed in `requirements.txt` if present.
2. Run your scripts or import the modules into your application.
